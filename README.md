# Project Portfolio

A multi-page personal portfolio website built with HTML, CSS, and JavaScript. The site introduces Sylvia, highlights projects, lists skills, shares education details, and provides a contact page.

## Pages

- Home
- Projects
- Skills
- Education
- Contact

## Features

- Multi-page site structure
- Shared navigation bar across pages
- Consistent styling with a single CSS file
- Project cards and call-to-action buttons
- Simple and beginner-friendly front-end structure

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

```text
Project-Portfolio/
|- index.html
|- projects.html
|- skills.html
|- education.html
|- contact.html
|- style.css
|- scripts.js
```

## How To Run

1. Clone or download the repository.
2. Open `index.html` in a web browser.

No build step or package installation is required.

## Purpose

This project serves as a personal portfolio website and a front-end practice project. It is useful for showcasing page layout, navigation, styling, and basic site organization.
